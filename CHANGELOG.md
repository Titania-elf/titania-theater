# 更新日志 / Changelog
## v3.0.9

- 🎨 **新增自定义主题样式**：设置中新增「主题样式」页，支持自定义 CSS 覆盖默认样式，提供常用选择器参考
- 🔧 **悬浮球透明度控制**：边框和背景色均支持 0-100% 透明度调节
- 📐 **渲染区域优化**：移除 Shadow DOM 内容区域默认内边距，生成内容可完全填充
- 🐛 **修复移动端分类重命名和批量移动剧本问题**

## v3.0.8
🎮 互动剧本增强：检测到互动内容时显示浮动按钮(FAB)，支持新窗口体验和导出HTML  
🎲 新增宏处理支持：剧本提示词现支持 {{random::A::B::C}} 等 ST 内置宏  
📂 剧本管理器优化：移除自定义排序，移动端新增下拉选择器  
🔧 修复批量移动剧本后原分类仍保留问题  
📊 审查窗口同步显示处理后的提示词

✍️ 自动续写增强：优化上下文拼接策略，智能去重，提升长文连贯性
